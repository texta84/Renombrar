# Renombrar archivos

Proyecto para renombrar o renombrar y copiar cualquier archivo, creado en lenguaje java.

## Modo de uso

- Utiliza el botón **BUSCAR** para agregar los archivos que vas a renombrar.
- Al agregar se mostraran los nombres de los archivos agregados en **LISTA DE ARCHIVOS**.
- Selecciona los nombres de los archivos que renombraras.
- Utiliza el botón **RENOMBRAR** o **RENOMBRAR COPIAR** para renombrar los archivos seleccionados.

> NOTA: Si quieres **RENOMBRAR Y COPIAR** debes habilitar la casilla **COPIAR**.
